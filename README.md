# ui-kit-utils

#### isTouch
`import isTouch from 'ui-kit-utils/is-touch'`

```javascript
import utils from 'ui-kit-utils'
var isTouch = utils.isTouch

isTouch() //- true | false
```


#### getOffset
`import getOffset from 'ui-kit-utils/get-offset'`

```javascript
import utils from 'ui-kit-utils'
var getOffset = utils.getOffset

getOffset(element) //- {top, left}
```
